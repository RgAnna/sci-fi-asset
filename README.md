# Sci-Fi Asset Pack: Дрон и Технический Блок для PvP

##  Характеристики ассетов

### Дрон (Разведка и Поддержка)
- **Полигоны:** 1,746 треугольников
- **Назначение:** Разведка, анализ местности, тактическая поддержка

- **Усиления (куб):**
